# Discord Bot Dashboard (Minimal)

هذا المشروع هو نسخة أولية لواجهة تحكم لبوت ديسكورد. يحتوي على خصائص أساسية لتخزين توكن البوت بشكل مشفّر، سجل نشاط بسيط، وواجهات API لإدارة البوتات.

الخطوات لتشغيل المشروع:

1. انسخ `.env.example` إلى `.env` وعيّن `MASTER_KEY` إلى قيمة HEX بطول 64 محرف (32 بايت).
2. ثبت الحزم:

```bash
npm install
```

3. شغّل التطبيق:

```bash
npm start
```

4. شغّل الاختبارات:

```bash
npm test
```

ملاحظة أمنيّة: احتفظ بـ `MASTER_KEY` في بيئة آمنة (مثل متغيرات بيئة الخادم أو secret manager). لا تخزنها في قاعدة البيانات.
